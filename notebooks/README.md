## 📁 Datasets Used

| Model                   | Dataset Name      | Description                                      |
| ----------------------- | ----------------- | ------------------------------------------------ |
| Linear Regression       | Quikr Car Dataset | Used for predicting car prices based on features |
| Logistic Regression     | Titanic Dataset   | Binary classification (Survived/Not Survived)    |
| Naive Bayes             | Titanic Dataset   | Classification using probabilistic approach      |
| Decision Tree           | Iris Dataset      | Multi-class classification of flowers            |
| KNN                     | Iris Dataset      | Classification based on nearest neighbors        |
| SVM                     | Iris Dataset      | Classification using hyperplanes                 |
| GridSearchCV            | Iris Dataset      | Hyperparameter tuning example                    |
| Random Forest           | Digits Dataset    | Classification of handwritten digits             |
| PCA                     | Digits Dataset    | Dimensionality reduction                         |
| K-Fold Cross Validation | Digits Dataset    | Model validation technique                       |
| Ensemble Learning       | Diabetes Dataset  | Regression using ensemble methods                |
